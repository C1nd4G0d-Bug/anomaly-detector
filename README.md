# 🛡️ Anomaly Detector (Blue Team Project)

Un script de seguridad en Python diseñado para analizar archivos de registro (`logs`) y detectar de forma automatizada posibles ataques de fuerza bruta.

## 🚀 Características
* Lectura y análisis automático de archivos `.log`.
* Detección de múltiples intentos fallidos de inicio de sesión.
* Seguimiento en memoria de direcciones IP atacantes.
* Alerta visual crítica en consola al superar el umbral de intentos permitidos.

## 🛠️ Cómo usarlo
Abre una terminal en el directorio del proyecto y ejecuta:

```bash
python detector.py
